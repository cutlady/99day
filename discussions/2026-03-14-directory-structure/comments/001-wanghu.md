## 讨论流程
1. **发起讨论**：创建讨论目录和README
2. **参与讨论**：在comments/添加评论文件
3. **形成结论**：讨论结束后创建conclusion.md
4. **状态更新**：README中状态改为closed

## 评论文件命名规则
```
comments/XXX-username.md
XXX：三位数字序号（001, 002, 003...）
username：评论人英文名（wanghu, wangdaxia, wangxiaoxie）
```

1. **完全可控**：无需GitHub特殊功能
